Workflow

  • 1. Webscrapping

    30,000 beer reviews, ratings and user information is scrapped from ratebeer.com

  • 2. Data Cleaning

    Text mining, tokenization, spell check and lemmatization was performed to process the review data

  • 3. Content-based Model

    Term Frequencyâ€“Inverse Document Frequency (TF-IDF) are calculated, Latent Semantic Indexing (LSI) was applied, and similarity matrix was generated according to cosine similarity

  • 4. Collaborative Filtering (SVD++)

    SVD++ is a derivative of low-rank matrix factorization technique. It further decomposes ratings to global average, user/item biases, implicit feedback, and latent features

  • 5. Collaborative Filtering (RBM)

    RBM is an unsupervised generative two-layer neural network consists of a visible and a hidden layer. Performs CF by reconstructing the user-item matrix.

  • 6. Build Recommendation Engine
